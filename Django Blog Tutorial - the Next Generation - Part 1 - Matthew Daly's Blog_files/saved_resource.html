                        Matthew Daly Mod A.A. • 6 months ago

                        The problem is that Postgresql isn't installed. You can install it using you package manager on most Linux distros. On Mac OS X Homebrew is your best bet. I can't advise on installing it on Windows as I never use it.

                        The tutorial was written with Django 1.6 in mind, and Django 1.7 has migrations baked in so South is redundant. I would suggest either looking at the documentation for Django 1.7's migrations and adapting the commands you run accordingly, or downgrading to 1.6.7 - once you're finished it's relatively straightforward to migrate over to 1.7 and move from South to the native migrations.
